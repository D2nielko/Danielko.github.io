## Binary search

Binary search is the canonical example of a divide-and-conquer algorithm. It
runs in $O(\log n)$ time on a sorted array by halving the search space on
each comparison.

```python
def binary_search(arr: list[int], target: int) -> int:
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target:
            return mid
        if arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
```


## A little math

Softmax turns a vector of real-valued scores into a probability distribution.
For a vector $z \in \mathbb{R}^n$:

$$
\text{softmax}(z)_i = \frac{e^{z_i}}{\sum_{j=1}^{n} e^{z_j}}
$$

It shows up everywhere from multiclass classifiers to attention weights in
transformers.
